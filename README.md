## Zielgruppe

### Geschäftskunden
###### Geschäftsführer 
```
Kleines bis mittelständiges Unternehmen
Preis wichtiger
Meist limitierteres technisches Verständnis
"Oberflächliche" Beratung
Auf letztendliche Vorteile fokussiert, an technischen Details weniger interessiert
-> Zeitersparnis, weniger frustrierte Mitarbeiter (Value of Life), Prozessoptimierung
----> Kosten-Nutzen
Achtet auf professionellen Auftritt, Zertifikate
Lässt sich auf Bling etwas ein, zuviel aber nix gut
Mittleres Alter, 40-65
Greift eher mit PC auf Seite zu
```
###### "Techies" - Technische Beauftragte
```
Meist Mitarbeiter oder "Beauftragter" -> Sohn von Ute
Fast ausschließlich Mittelständler
Meist etwas umfangreicheres technisches Verständnis, bishin zu professionellen Qualifikationen
Eher produktbezogene, technische Beratung. Technischer Backround relevant
Eher auf saubere, sinnvolle Lösung fokussiert
Achtet auf gebotene Dienstleistungen, hat meist schon ungefähre bis genaue Vorstellungen der gewünschten Dienstleistung
Achtet auch auf Zertifizierungen 
Appreciatet moderne Webauftritte und Gimmicks
Lassen sich weniger von typischer Endnutzer-Designsprache beeindrucken
Marketinggewäsch = Fettnäpfchen
Alter 20-40
Oft weniger förmlich
Greift fast ausschließlich mit PC auf Seite zu
```
